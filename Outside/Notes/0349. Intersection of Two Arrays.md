### Remark
- 只是要找出相同的值而已
- 有比對到就可以刪除掉了
---
### Code
```python
class Solution:
    def intersection(self, nums1: List[int], nums2: List[int]) -> List[int]:
        hashmap = {}
        res = []
        for i in nums1:
            hashmap[i] = hashmap.get(i, 0) +1
        for i in nums2:
            if i in hashmap:
                res.append(i)
                del hashmap[i]
        return res
```
