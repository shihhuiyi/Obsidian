### Remark
- 有序array平方後最大的會在兩端，把指針放在兩端
- i這邊也可以用-1
---
### Code
```python
class Solution:
    def sortedSquares(self, nums: List[int]) -> List[int]:
        left, right, i = 0, len(nums)-1, len(nums)-1
        res = [0] * len(nums)
        while left <= right:
            if nums[left] ** 2 > nums[right] ** 2:
                res[i] = nums[left] ** 2
                i -= 1
                left += 1
            else:
                res[i] = nums[right] ** 2
                i -= 1
                right -= 1
        return res
```

### Complexity
- 時間複雜度：O(n)
- 空間複雜度：O(1)
