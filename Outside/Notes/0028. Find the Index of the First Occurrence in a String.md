### Remark
- 直接用find找
	- python find不是用KMP，是用其他更大跳躍的演算法
- 用KMP找
---
### Code
```python
class Solution:
    def strStr(self, haystack: str, needle: str) -> int:
        return haystack.find(needle)
```
---
### Complexity
- 時間複雜度：O()
- 空間複雜度：O()
