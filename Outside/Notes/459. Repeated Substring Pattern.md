### Remark
- KMP解法
- 重複字串解法
	- 如果原字串是由重複子字串組成的，代表至少由2個子字串組成
	- 把兩個原字串合併，並去掉頭尾，先破壞原本的字串
	- 如果頭被破壞的這個子字串可以由下一個組成的裡面補上
	- 代表是由重複字串所組成的
---
### Code
```python
class Solution:
    def repeatedSubstringPattern(self, s: str) -> bool:
        return s in s[1:]+s[:-1]
```
