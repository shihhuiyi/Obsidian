### Remark
> 該用set的時候還是得用
- 用set紀錄算過的值，避免無限循環
- 迭代新的n
---
### Code
```python
class Solution:
    def isHappy(self, n: int) -> bool:
        seen = set()
        while n != 1:
            if n in seen:
                return False 
            seen.add(n)
            n = sum(int(i) ** 2 for i in str(n))
        return True
```

### Complexity
- 時間複雜度：O()
