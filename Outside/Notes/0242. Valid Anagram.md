### Remark
- dict紀錄集合和數量
- 比對兩個dict相不相同
---
### Code
```python
class Solution:
    def isAnagram(self, s: str, t: str) -> bool:
        dict_s, dict_t = {}, {}
        for i in s:
            dict_s[i] = dict_s.get(i, 0) +1
        for i in t:
            dict_t[i] = dict_t.get(i, 0) +1
        return dict_s == dict_t
```
---
### Complexity
- 時間複雜度：O(n)
- 空間複雜度：O(1)
