### Code
```python
class Solution:
    def removeElement(self, nums: List[int], val: int) -> int:
        slow, fast = 0, 0
        while fast <= len(nums) -1:
            if nums[fast] != val:
                nums[slow] = nums[fast]
                slow += 1
            fast += 1
        
        return slow 
```
---
### Complexity
- 時間複雜度：O(n)
- 空間複雜度：O(1)
