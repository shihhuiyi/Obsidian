### Remark
- 把所有字反轉一次，在把單字反轉，也可以達到一樣的效果
---
### Code
```python
class Solution:
    def reverseWords(self, s: str) -> str:
        return " ".join(i for i in s.split(" ")[::-1] if i != "")
```
---
### Complexity
- 時間複雜度：O(n)
- 空間複雜度：O(n)
