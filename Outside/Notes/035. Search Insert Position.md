### Remark
- 如果找不到target就返回要插入的目標
- target區間的最左邊就會是要插入的點 -> return left
---
### Code
```python
class Solution:
    def searchInsert(self, nums: List[int], target: int) -> int:
        left, right = 0, len(nums)-1
        while left <= right:
            mid = left + (right - left) // 2
            if nums[mid] == target:
                return mid
            elif nums[mid] > target:
                right = mid -1
            else:
                left = mid +1

        return left
```
---
### Complexity
- 時間複雜度：O(log n)
- 空間複雜度：O(1)
