### Remark
- 用two pointer來做翻轉
- curr會紀錄下個點，所以要return pre才對
---
### Code
```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def reverseList(self, head: Optional[ListNode]) -> Optional[ListNode]:
        curr, pre = head, None
        while curr:
            temp = curr.next
            curr.next = pre
            pre = curr
            curr = temp
        return pre
```
---
### Complexity
- 時間複雜度：O(n)
- 空間複雜度：O(1)
