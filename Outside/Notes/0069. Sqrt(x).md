### Remark
- 想像是在0到x之間做搜尋
- 要找的是平方後<=x的 -> return left -1
---
### Code
```python
class Solution:
    def mySqrt(self, x: int) -> int:
        left, right = 0, x
        while left <= right:
            mid = left + (right - left) // 2
            if mid ** 2 == x:
                return mid
            elif mid ** 2 > x:
                right = mid -1
            else:
                left = mid + 1
        
        return left -1
```
---
### Complexity
- 時間複雜度：O(log n)
- 空間複雜度：O(1)
