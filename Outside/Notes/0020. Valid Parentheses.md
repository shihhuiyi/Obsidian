### Remark
- dict用法也可以用dict("()", "[]", "{}")
- 要開始判斷的時候要注意stack有沒有東西
- return的時候也要注意stack有沒有東西，有東西代表不是正確的用法
---
### Code
```python
class Solution:
    def isValid(self, s: str) -> bool:
        hashmap = {"(": ")", "[": "]", "{": "}"}
        stack = []
        for i in s:
            if i in hashmap:
                stack.append(i)
            else:
                if not stack or i != hashmap[stack.pop()]:
                    return False
        return not stack 
```
---
### Complexity
- 時間複雜度：O(n)
- 空間複雜度：O(1)
