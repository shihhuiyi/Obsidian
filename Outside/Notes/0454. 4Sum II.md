### Remark
- 跟015三數和、018四數和不同
	- 這兩題用hashmap很難不超時
- 找出加起來=0的組數
	- 用四個迴圈: O(n^4)
	- 一個迴圈+三個迴圈: O(n^3)
	- 二個迴圈+兩個迴圈: O(n^2)
- count要+之前組合出現的次數
---
### Code
```python
class Solution:
    def fourSumCount(self, nums1: List[int], nums2: List[int], nums3: List[int], nums4: List[int]) -> int:
        hashmap = {}
        count = 0
        for i in nums1:
            for j in nums2:
                hashmap[i+j] = hashmap.get(i+j, 0) +1

        for i in nums3:
            for j in nums4:
                if -(i+j) in hashmap:
                    count += hashmap[-(i+j)]
        return count
```
---
### Complexity
- 時間複雜度：O(n^2)
- 空間複雜度：O(n^2)
