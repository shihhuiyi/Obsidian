### Remark
- 如果過去有target-現在的值，代表找到了
---
### Code
```python
class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        hashmap = {}
        for idx, n in enumerate(nums):
            if target - n in hashmap:
                return [hashmap[target-n], idx]
            hashmap[n] = idx
```
