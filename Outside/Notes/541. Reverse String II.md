### Remark
- 不是在考反轉，用list[::-1]比較快
- 每2k才要反轉一次，回圈每次+2k速度更快
---
### Code
```python
class Solution:
    def reverseStr(self, s: str, k: int) -> str:
        idx = 0
        while idx <= len(s)-1:
            idx_k = idx + k
            s = s[:idx] + s[idx:idx_k][::-1] + s[idx_k:]
            idx = idx + k*2
        return s
```
---
### Complexity
- 時間複雜度：O(n)
- 空間複雜度：O(1)
