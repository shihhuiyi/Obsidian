### Remark
- sliding window
	- 窗口是count數量<=2
	- 還沒超過就動終點
	- 超過就動起點
- 用hashmap紀錄果樹和數量
	- 要判斷數量，所以變0要移除掉
---
### Code
```python
class Solution:
    def totalFruit(self, fruits: List[int]) -> int:
        left, right = 0, 0
        count = {}
        max_len = 0
        while right <= len(fruits)-1:
            count[fruits[right]] = count.get(fruits[right], 0) +1
            while len(count) >2:
                count[fruits[left]] -= 1
                if count[fruits[left]] == 0: del count[fruits[left]]
                left += 1
            max_len = max(max_len, right-left+1)
            right += 1
        return max_len
```

### Complexity
- 時間複雜度：O(n)
- 空間複雜度：O(1)
