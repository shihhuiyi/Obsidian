### Remark
> 匹配問題都是stack的強項
- 如果stack裡面發現重複的就移除掉
- 否則就加進去stack裡面
---
### Code
```python
class Solution:
    def removeDuplicates(self, s: str) -> str:
        stack = []
        for i in s:
            if stack and i == stack[-1]:
                stack.pop()
            else:
                stack.append(i)
        return "".join(stack)
```
---
### Complexity
- 時間複雜度：O(n)
- 空間複雜度：O(1)
