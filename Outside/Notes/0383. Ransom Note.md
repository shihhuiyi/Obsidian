### Code
```python
class Solution:
    def canConstruct(self, ransomNote: str, magazine: str) -> bool:
        hashmap = {}
        for i in magazine:
            hashmap[i] = hashmap.get(i, 0) +1
        for i in ransomNote:
            if i not in hashmap:
                return False
            hashmap[i] -= 1
            if hashmap[i] ==0:
                del hashmap[i]
        return True
```
---
### Complexity
- 時間複雜度：O(n)
- 空間複雜度：O(1)
